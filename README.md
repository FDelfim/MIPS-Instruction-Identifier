# MIPS Instruction Identifier

## Implement a Java program to recognize each of the following instructions: `add`, `lw`, `memory`, `subtract`, `load word`, `store word`.

### Example Code:

- `memory[0]`
- `add $t1,$s4,$s4`
- `add $t1,$t1,$t1`
- `sub $t1,$t1,$s3`
- `lw $t0,0($t1)`
- `add $s1,$s2,$t0`
- `sw $t0,1200($t1)`
