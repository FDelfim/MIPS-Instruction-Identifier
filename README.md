# Exercício Compiladores - Identificador de instruções MIPS

<h3>Implemente um programa em Java para reconhecer cada uma das seguintes instruções add, lw, memory, subtract, load word, store word.</h3>

Exemplo de Código:

memory[0]

add $t1,$s4,$s4

add $t1,$t1,$t1

sub  $t1,$t1,$s3

lw $t0,0($t1)

add $s1,$s2,$t0

sw $t0,1200($t1)
